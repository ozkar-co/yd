# YouTube CLI Downloader

Script para buscar y descargar videos de YouTube desde la terminal.

## Requisitos
- Python 3.7+
- FFmpeg instalado en el sistema.

## Uso
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `python script.py`
3. Ingresar término de búsqueda.
4. Seleccionar un índice (1-9) para descargar.

## Características
- Límite de resolución: 720p.
- Audio: Calidad balanceada (128kbps).
- Formato: MP4.
